# Mini-Cloud

> 如果它看起来像MINI，开起来像MINI，那么它可能就是一台MINI

> 微信小程序简单HTTPS接口，基于gin实现，使用MySQL持久化

## 安装部署

### go安装

#### gin安装

> go get -u github.com/gin-gonic/gin

> go get github.com/kardianos/govendor

> govendor init

> govendor fetch github.com/gin-gonic/gin@v1.3

#### 依赖包

- nginx

- HTTPS

- 图片分离

- MySQL
